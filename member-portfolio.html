歡迎觀看作品集

以下包含Android App、RWD網站、美食地圖、個人履歷的簡短介紹,當中有下載或觀看連結與GitHub,另外有需QR Code請點擊履歷表並下滑到底即可.

Android App

使用政府開放數據提供相關美食資料,讓使用者可查詢在地美食餐廳,並可以收集與移除.

  • Google Sign-In
  • Open Data
  • MySQL/SQLite
下載連結 GitHub
網頁設計

採用SPA與RWD設計網頁並開發會員系統,可使用第三方登入,會員可更改密碼,另外有管理員身分可對所有會員做相關設定.

  • SPA
  • RWD
  • 會員系統/CRUD
觀看網站 GitHub
美食地圖

使用政府開放數據提供相關美食資料,選擇縣市鄉鎮後會在左側清單顯示該地區資料並在地圖透過水滴顯示餐廳位置.

  • Open Data
  • Leaflet
